1. Open BurgerOrder in your ubuntu therminal

2. Install pip for python if you haven't done that already:
sudo apt install python3-pip

3. Install flask so your able to run the program:
pip install Flask

4. Install our requirements.txt file so you get the same flask build as the project is using:
pip install -r requirements.txt

5.Install SQLAlchemy so the program can run the database:
pip install sqlalchemy

6. Run this command to start the virtual environment:
source venv/bin/activate

7. Run this command to start the website on the virtual environment using our written code:
flask --app Containers/KitchenView.py run

8. Copy the given IP:GATE into your browser and start using the website :D
